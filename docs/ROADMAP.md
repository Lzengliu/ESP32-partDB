# 后续路线

版本：V1.1

## 优先级高

- 为 Web 页面和管理 API 加入认证、CSRF 防护与危险操作权限分级。
- 增加 Part-DB mock/API 回归测试及 OTA 自动回归。
- 增加详情长文本滚动和多批次选择页。
- 建立固定二维码图像集，对 ZXing/quirc、曝光和焦距变化做回归。

## 优先级中

- 为正式 PCB 分配独立 PN532 reset，移除 GPIO0 临时复位接线。
- 把内嵌 Web HTML/CSS/JS 拆成独立可测试资源构建流程。
- 为上传字体增加预生成字库流程，或集成受控的运行时栅格化。
- 实现 GIF/WebP 开机动画逐帧播放。
- 增加离线详情缓存与 TF 操作日志。

## 长期

- 多 Part-DB 实例、用户权限与操作审计。
- 标签打印机、外部扫码枪和更多 NDEF 记录类型。
- 多屏幕/触摸/相机硬件抽象。
- 自动化硬件测试夹具。
