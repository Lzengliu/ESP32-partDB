# V1.0 到 V1.1 变更对比

[English Version](CHANGES_V1.0_TO_V1.1_EN.md)

对比基线：`dist/v1.0/source/` 保存的 V1.0 公开源码包。

## 不兼容变更

| 项目 | V1.0 | V1.1 |
| --- | --- | --- |
| 应用分区 | factory/ota_0/ota_1 各 `0x300000` | 各 `0x480000` |
| coredump 起始 | `0x920000` | `0xDA0000` |
| fontfs | `0x2E0000` | `0x100000` |
| spiffs | `0x3E0000` | `0x140000` |
| 首次升级方式 | 原布局 | 必须刷 V1.1 merged 镜像到 `0x0` |

V1.0 Web OTA 只替换应用，不会替换分区表，因此不能用于第一次升级到 V1.1。

## 硬件和稳定性

- 启用 8 MB Octal PSRAM，并增加分配/容量诊断。
- 触摸总线移至 `GPIO3/GPIO44`，避开 PSRAM/MSPI 引脚。
- PN532 使用独立硬件 I2C `GPIO43/GPIO41`，增加 IRQ、恢复和后台服务。
- Flash 频率调整为 40 MHz，日志切换到 USB Serial/JTAG。
- 增加外设仲裁，减少相机、NFC 等耗时操作冲突。

## UI 和资源

- 重构主页、搜索结果、详情、快捷、信息与设置页面。
- 增加自动息屏、启动图、背景/锁屏资源管理和更多诊断状态。
- 新增 8/12 px 字模，保留 16 px 字模，改善高密度中文显示。
- 搜索结果可直接按 Part ID 进入详情，减少重复精准查询。

## Part-DB 和库存

- 增加搜索/详情缓存及缓存来源状态。
- 扩展 Part、Lot、IPN、条码和 NFC 内容路由。
- 完善详情、库存批次和二进制诊断上传路径。

## 相机和扫码

- 相机帧使用 PSRAM，扫码使用 SVGA 灰度帧。
- 新增本地 ZXing-C++ 快速解码，保留 quirc 回退。
- 增加自适应曝光稳定帧和条件式二次扫描。
- 移除无效 AF 入口；当前模组只支持手动调焦。

## NFC

- 新增 PN532 独立 I2C、总线恢复、固件探测与状态缓存。
- 新增后台读卡、NDEF 读取/写入/清除和 Part-DB 路由。
- 明确 GPIO0 仅为原型恢复复位的临时方案。

## Web、OTA 和维护

- Web 状态增加 PSRAM、NFC、相机、扫码、UI、作者和仓库信息。
- 增加 TF 文件/资源管理、相机诊断和更完整的配置入口。
- OTA 增加空包、容量校验以及启动健康确认，修复启用 rollback 后新镜像未确认可能回滚的问题。
- 上传分配、接收或写盘失败时清理残留的不完整文件。
- 默认设备密钥缺失时会生成并持久化；AP 日志不再输出密码。
- 增加 HTTP socket 余量、空闲连接 LRU 清理和 Part-DB 请求互斥，避免 Web 长时间使用后拒绝新连接。

## 发布工程

- 项目版本改为 `1.1.0`。
- 补齐作者“灵异大队长”和开源地址。
- 补齐 ZXing-C++、quirc、GNU Unifont、江城斜黑体及 OFL 声明。
- 新增可重复打包脚本、完整固件包、源码包和 SHA-256 校验流程。
